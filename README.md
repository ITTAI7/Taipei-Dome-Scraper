# Taipei Dome Scraper (大巨蛋售票極速查詢系統)
專門爬取各大售票網（目前支援 ibon 的樂天、統一獅）大巨蛋賽事剩餘票數的微服務。

## 如何啟動伺服器
1. 安裝依賴套件：`npm install`
2. 開啟熱更新伺服器：`npx tsx --watch server.ts`
